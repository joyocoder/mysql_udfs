mysql_udfs by joyocoder

MySQL UDF collection of Jo Yo; JavaScript in MySQL (JsMap).

This is a collection of UDFs. The most important (and recent addition) is JsMap that allows JavaScript to be evaluated on an object to do... stuff.

Dependencies

SpiderMonkey, MySQL

Install

There are some manual steps and platform integration is possible provided it is Linux. There is a wiki on how to install it, and it needs to be cleaned up.
